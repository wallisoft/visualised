Quick Start

Linux (Debian/Ubuntu)

# Install .NET 9
sudo apt install dotnet-sdk-9.0

# Clone and run
git clone https://github.com/wallisoft/visualised.git
cd visualised
dotnet build
dotnet run -- --designer
